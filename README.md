# Heart & Diabetes Awareness System

## Project Description

This project is an educational web application designed to increase awareness about the relationship between physical activity, heart health, and diabetes prevention.

The system provides structured medical information, health indicators, and interactive tools to help users understand the importance of exercise in reducing the risk of heart disease and diabetes.

---

## Project Objectives

- Increase awareness about heart health
- Explain causes and symptoms of diabetes
- Highlight the role of physical activity in disease prevention
- Provide BMI calculator
- Provide Heart Risk indicator
- Provide Blood Sugar level analysis tool
- Present medical information in an interactive dashboard format

---

## Technologies Used

- React.js
- Tailwind CSS
- React Router
- Chart.js
- Framer Motion
- Web Speech API (Arabic voice support)

---

## Project Features

- Interactive medical dashboard
- Heart health awareness section
- Diabetes awareness section
- BMI calculator
- Heart risk indicator
- Blood sugar analyzer
- Charts for health visualization
- Arabic voice reading support
- Animated UI components
